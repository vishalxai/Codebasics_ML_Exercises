# Outlier Detection using IQR and Box Plot

This repository contains a Jupyter Notebook that demonstrates how to detect and handle outliers in a dataset using the **Interquartile Range (IQR) method** and visualize them with **box plots**.

---

## 📌 Contents
- Load and inspect datasets (`heights.csv`, `region_wise_sales.xlsx`)
- Compute **Q1, Q3, and IQR**
- Calculate lower and upper bounds for outlier detection
- Identify and remove outliers
- Apply IQR method on **heights data**
- Apply IQR method on **sales data (by region)**
- Visualize results using **box plots**

---

- Python 3
- Pandas
- Numpy
- Matplotlib
